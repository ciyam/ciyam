Installing the CIYAM "systemd" Service
--------------------------------------

NOTE: Do this after the CIYAM application server has been
successfully built and after the Meta DB has been created.

sudo cp ciyamd.service /etc/systemd/system

sudo vi /etc/systemd/system/ciyamd.service
(edit ciyamd.service file to change "tux" to the appropriate user)

sudo systemctl daemon-reload
sudo systemctl enable ciyamd

service ciyamd start
service ciyamd status
